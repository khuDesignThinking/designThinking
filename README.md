git 쓰는 방법<br>
git status : 상태 확인<br>
바뀐 파일은 git status 치면, 빨간색<br>
git add -A // 전체 한번 묶음. // git status 치면, 초록색<br>
git commit -m"할말" //git status치면, 1개를 push할 수 있다고 나옴. <br>
git push // 서버에 올리는 것임. 하기 전에, git pull 해야함<br>
git pull // 서버에서 다운 받기<br>
